Descreva, em suas palavras, o que acontece quando tentamos acessar um site?

Quando o cliente acessa um site uma solicitação é enviada para o servidor que hospeda aquele site, fazendo com que o servidor reune todas as informações que tem,
novamente ocorre uma outra resposta entre servidor e navegador diversas vezes até que finalmente a pagina final seja mostrada para o usário.
