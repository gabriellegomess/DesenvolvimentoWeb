Organização com o FlexBox (flex-grow)

Podemos separar o espaço de maneira desigual para cada um de nossos elementos

Comportamento padrão

Por padrão, todos os elementos de um conteiner recebem o mesmo espaço

1

2

3

Aumentando o peso de 1

Podemos aumentar o espaço ocupado pelo primeiro elemento

Nesse caso, 1 recebeu flex-grow: 1

1

2

3

.grow_1 {
    flex-grow: 1;
}

Aumentando ainda mais o espaço

Podemos aumentar ainda mais o espaço ocupado por um elemento (se fizer sentido/nesse caso não faz...).

Nesse caso, 1 recebeu flex-grow: 2

1

2

3

.grow_2 {
    flex-grow: 2;
}

Aumentar o peso de dois

Podemos ainda aumentar o espaço de vários elementos.

Nesse caso, o 1 e o 3 apresentam flex-grow: 1

1

2

3

.grow_1 {
    flex-grow: 1;
}

Pesos diferentes

Podemos ainda colocar espaços diferentes em cada elemento

Nesse caso, o 1 apresenta flex-grow: 1 e o 3 apresentam flex-grow: 3

1

2

3

.grow_1 {
    flex-grow: 1;
}
