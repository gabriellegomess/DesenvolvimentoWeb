HTML é a linguagem padrão para construção de páginas web.

Com HTML é possivel definir o conteudo e a construção do documento

Tags em HTML permitem a criação de links, imagens e tabelas.

JavaScript é uma linguagem de programação usada para criar interatividade na web.

CSS é uma linguagem usada para estilizar documentos HTML.

Com CSS, é possivel controlar a aparência e o layout da página
