Flexbox

O modelo de layout flexbox permite que oganizemos elementos (denominados Flex-items) dentro de um container (Flex-container).

Esta página permite observar o funcionamento de algumas propriedades do flexbox.

Flex-direction

A propriedade flex-direction permite definir a direção em que os elementos serão posicionados dentro do elemento pai.

Posicionamentos

Existem 3 propriedades importantes para o posicionamento dos flex-items dentro de um flex-container. Para que essas propriedades façam diferença, o pai (container) precisa ser maior que o tamanho necessário para os elementos filhos.

  • justify-content permite alinhar nossos elementos horizontalmente.
  • align-content permite alinhar nossos elementos verticalmente, necessita da propriedade flex-wrap: wrap;.
  • align-items permite alinhar nossos elementos em relação ao bloco de outros, por exemplo, deixar todos alinhados em baixo, ou em cima.

Multilinha

Como você deve ter percebido, a propriedade align-content precisa que o flex-wrap: wrap; esteja ativo para funcionar. Isso porque essa propriedade é planejada para conteúdos com várias linhas. (Observe que align-content: stretch não fazia nada)

  • justify-content permite alinhar nossos elementos horizontalmente.
  • align-content permite alinhar nossos elementos verticalmente, necessita da propriedade flex-wrap: wrap;.
  • align-items permite alinhar nossos elementos em relação ao bloco de outros, por exemplo, deixar todos alinhados em baixo, ou em cima.
